## Secrets (1Password)

- **CLI:** `op`
- **Service account token:** stored in `~/.zshenv` as `OP_SERVICE_ACCOUNT_TOKEN`
- **App usage:** `op run --env-file .env.op -- <command>`
- **Script usage:** `op read "op://vault/item/field"`

### Vault Structure
- **Master vault** (`lobsterfarm`): Shared service credentials (Vercel, Sentry, domain registrar, etc.) — services that manage multiple entities under one account.
- **Per-entity vaults** (`entity-{id}`): Entity-specific secrets (API keys, database credentials, third-party integrations unique to that entity).
