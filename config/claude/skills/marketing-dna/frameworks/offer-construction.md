# Offer Construction — Cross-Author Synthesis

How Hormozi, Suby, and Brunson all build an offer. Same animal, three vocabularies.

## The synthesis recipe

Every great offer has SEVEN ingredients. Each ingredient maps to multiple authors:

### 1. Dream Outcome (Hormozi) / Big Promise (Schwartz) / Outcome (Brunson)

The end state the prospect imagines. Not the product, the FEELING.

CombatCall: "A green Sunday ledger. The screenshot you finally get to post."

### 2. Mechanism (Schwartz) / Vehicle (Brunson)

The HOW. The differentiator. The "why this works when nothing else did."

CombatCall: "Power-accuracy-weighted +EV scoring across every UFC fight since 2010. Three independent models — striking, grappling, situational — combined via Bayesian weight."

### 3. Stack of value (Hormozi) / Stacked Value (Brunson) / Godfather components (Suby)

Itemize what they get. Each item with a perceived dollar value. Sum > price.

CombatCall:
- Verified pick of the week, every event ($600/yr if from a tout)
- Full fighter dossier per matchup ($240)
- Edge-score model output, transparent reasoning ($300)
- Bankroll-management guide PDF ($97)
- Historical pick archive (every event, audited) ($200)
- Discord community access (FUTURE — $300)
- **Total stacked: $1,737/yr. Today: $120/yr.**

### 4. Bonuses that solve objections (Hormozi)

Each bonus targets a specific objection the prospect has.

CombatCall objections + bonus answers:
- "Pick services are scams" → "Public W/L ledger, every pick, every event."
- "I don't have a big bankroll" → "Bankroll-sizing calculator + +EV bet sizing guide."
- "I don't know UFC well enough" → "Fighter dossier in plain English."
- "What if I miss a card?" → "Pick alert via email + push, 4hr before bell."

### 5. Risk reversal (Hormozi / Suby) / Guarantee

Strongest form: anti-guarantee or implied. Realistic for CombatCall: conditional.

CombatCall: "Cancel anytime, no questions. Annual plan: prorated refund if you cancel within 30 days." (Subscription SaaS standard. Don't promise gambling outcomes — regulatory.)

### 6. Scarcity / Urgency (Hormozi / Cialdini)

Must be REAL.

CombatCall: "Founding member rate ($120/yr) locks for first 500 subscribers. After: $150/yr."

### 7. Naming (Hormozi MAGIC formula)

See `authors/hormozi/frameworks.md` §3.

CombatCall offer names to test:
- "CombatCall +EV Pro" (vanilla, fine)
- "The Fight Intelligence Terminal" (positions a new category — Brunson "new opportunity" play)
- "The UFC Bettor's Edge System" (Avatar + Goal + Container — MAGIC formula)

## The Build Sequence

1. Start with the prospect's pain (Robbins). Open the wound.
2. Reveal the dream outcome (Hormozi). Paint the future.
3. Introduce the mechanism (Schwartz). Explain WHY this is different.
4. Stack the value (Hormozi + Brunson). Item by item.
5. Crush objections with bonuses (Hormozi). Each one answers a "but what about..."
6. Reverse the risk (Hormozi). Guarantee.
7. Apply urgency (Cialdini, real). Scarcity.
8. Make the ask (Suby). Clear, single, action.

## Pricing — the offer-math (Hormozi)

Hormozi's pricing heuristic: price = 1/10 of the stacked value, OR price < 1/3 of the dream outcome's economic value. Whichever's higher.

CombatCall: $120/yr annual.
- 1/10 of $1,734 stacked = $173. So $120 is BELOW Hormozi's "anchor low" — could push to $149-179 if conversion is strong.
- Dream outcome economic value: even modest bettor profits $500-$2000/yr from better picks → $120 << $500 → check.
- Verdict: $120/yr is well-priced for the freemium funnel right now. Test $149/yr later, after social proof builds.
