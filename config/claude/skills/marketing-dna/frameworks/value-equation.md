# Value Equation — Cross-Author Synthesis

## The core (Hormozi)

```
                Dream Outcome × Perceived Likelihood
  Value  =  ───────────────────────────────────────────
                Time Delay × Effort & Sacrifice
```

Numerator builds value. Denominator destroys it. Most marketers obsess over the top. The leverage is usually below the line.

See `authors/hormozi/frameworks.md` §1 for the full breakdown.

## How other authors talk about the same thing

**Sabri Suby — the Godfather Offer.** Same math, different language. Suby's Godfather is the Value Equation expressed as a stacked offer with risk reversal. Where Hormozi is mathematical, Suby is theatrical — same idea, more sales-page-coded.

**Brunson — Hook-Story-Offer.** The "Offer" component is Brunson shorthand for what Hormozi calls a Grand Slam. Brunson stacks value the same way; he just bookends with story.

**Schwartz — Mass Desire.** Schwartz reminds you that the Dream Outcome isn't whatever YOU say it is. It has to map to a desire that already exists in the prospect's head. Many "good offers" fail because they're optimizing the math against the wrong desire.

**Cialdini — Risk Reversal as Reciprocity.** Hormozi's guarantees increase Perceived Likelihood; Cialdini explains *why* — the seller assuming the risk first triggers reciprocity, which lowers prospect resistance independent of the literal expected value.

**Robbins — Pain-Pleasure as Dream Outcome.** Robbins reminds you that the "outcome" the prospect cares about is emotional, not factual. The Dream Outcome is a STATE, not an event.

## How to actually use the Value Equation

1. **Write down the equation.** All four variables, in plain language.
2. **Score each 1-10 from the prospect's POV.** Be brutal.
3. **Find the lowest-scoring variable.** That's where the leverage is.
4. **Brainstorm 5 moves to push that variable up.**
5. **Implement the cheapest one first.**

## CombatCall worked example

| Variable | Score | Why | Move |
|---|---|---|---|
| Dream Outcome | 7/10 | "Win more UFC bets" is clear, but undersold on the landing page. | Sharpen hero copy. Make the dream visceral — "the green Sunday ledger." |
| Perceived Likelihood | 4/10 | NO verified track record visible to a stranger. Biggest gap. | Public-facing W/L tracker. Last-card recap email. Specific screenshots. Mechanism reveal. |
| Time Delay | 9/10 | Free 3 fights → immediate. Next event → 7 days max. Already strong. | Lean into it. "Saturday tells you." |
| Effort & Sacrifice | 8/10 | Read the card, place bets. Already low. | Stay low. Don't add a course/onboarding burden to the core offer. |

**Conclusion.** Perceived Likelihood is the dominant lever. Spend 80% of marketing investment there.
