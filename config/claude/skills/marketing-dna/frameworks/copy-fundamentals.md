# Copy Fundamentals — Cross-Author Synthesis

Schwartz on headlines. Hormozi on hooks. Suby on body copy. The complete view of "how to write a sales asset."

## The Headline

**Schwartz's rule.** The headline's only job is to make them read the second sentence. Match it to the prospect's awareness stage.

**Hormozi's rule (hooks).** The first 3 seconds determines everything in attention-starved media. Use a HOOK — not a clever turn of phrase, but a thing that stops the scroll. Curiosity + relevance.

**Suby's rule.** Pre-headline (small text, sets avatar) + headline (big promise + mechanism + dream outcome).

**The hybrid template.**

```
[Pre-head: AVATAR + situation]
[HEADLINE: big promise + curiosity hook + (optional) mechanism teaser]
[Sub-head: amplifies + adds proof]
```

CombatCall worked example:

```
Pre-head:  Attention: UFC bettors tired of buying picks that don't work.
HEADLINE:  How a fed-up bettor stopped trusting tipsters and built his own UFC model — and why he's giving the first three picks away free, every card.
Sub-head:  Trained on every UFC fight since 2010. Used by Cole + a small crew of bettors so far. Three free picks every event, no card required.
```

That headline does:
- Curiosity (the spreadsheet story).
- Specific mechanism (own model trained on every fight since 2010 = credible specificity, Schwartz Stage 5 identification).
- Mechanism teaser (every fight since 2010).
- Risk reversal (free, no card).
- Identification (broke bettor = the prospect).

## The Lead (first 100-200 words after headline)

**Schwartz's rule.** The lead is where you either WIN or LOSE the prospect. It should hit the strongest desire in the market, prove the headline isn't a tease, and create the slide into the rest of the page.

**Brunson's rule.** Open with story (Epiphany Bridge). Hook → first beat of the origin.

**Suby's rule.** Open the wound. Agitate the pain. Make them nod "yes that's me."

**The hybrid template.**

```
[ONE-SENTENCE STORY HOOK that matches the headline's promise]
[THE PROBLEM — agitate, make tangible, use prospect's own language]
[THE FAILED ALTERNATIVES — what they've already tried, why it didn't work]
[THE PIVOT — "but then I found / built / discovered..."]
```

CombatCall worked example:

```
I (Cole) lost money betting on UFC for years.

Three or four months ago I got tired of it and started building my own model.
Scores every fight on the same data the books use to set the lines —
power-accuracy, grappling control, situational edge. Trained on every UFC
fight since 2010.
```

## The Mechanism Reveal

**Schwartz's rule.** Stage 3-4 markets demand mechanism. The HOW. The specific reason your product works when others didn't.

**Hormozi's rule.** The mechanism is the credibility lever. The more specific, the more believable.

**The template.**

```
[ONE-SENTENCE MECHANISM CLAIM in plain English]
[2-3 SPECIFIC COMPONENTS broken out]
[VISUAL or PROOF — a chart, a sample output, a screenshot]
[WHAT IT'S NOT — distinguishes from competitor mechanisms]
```

CombatCall worked example:

```
The CombatCall model is three independent UFC-specific scoring systems combined.

  1. STRIKING — power-accuracy weighted, decayed by recency, adjusted for opponent caliber.
  2. GRAPPLING — control-time × submission-attempt rate, normalized to weight class.
  3. SITUATIONAL — camp changes, weight cuts, ring rust, cage size, prelim vs main.

Each fight gets three scores. We weight them by historical predictive power for THAT matchup type.
Then we cross-reference with current odds for +EV. That's the pick.

It's NOT a power ranking. NOT a striking-only metric. NOT trained on the last 6 months only.
Every UFC fight, every weight class, since 2010. ~5,200 fights of training data.
```

## The Body Copy Rules

**Schwartz:** every paragraph proves the headline. No new claims without proof. Body copy = evidence.

**Hormozi:** plain-spoken. Short sentences. White space. Read-aloud test — does it sound like a human?

**Suby:** long copy works when it has rhythm. Mix sentence lengths. Use bullets to break up density. Bold the punchlines.

**Brunson:** stories + analogies. Make the abstract concrete. Repeat the Big Domino throughout.

**Universal rules.**
- Active voice.
- "You" early and often (Schwartz technique #1 of the 38).
- Specific > vague (numbers, names, dates).
- Cut every sentence by 30%, then read it again.
- Read it aloud. If you stumble, the prospect will too.

## The CTA

**Hormozi:** one CTA per page. Repeat it. Make it tactile — "Click here," "Tap below," "Get started" with the action verb visible.

**Suby:** the CTA section is its own micro-pitch. Restate the offer. Restate the guarantee. Restate the price + value. THEN the button.

**Brunson:** the CTA is consistent through the page. Same button text every time.

**The template.**

```
[Restate offer in one sentence]
[Restate the stack in 2-3 line bullets]
[Restate the guarantee in one sentence]
[Restate the price + bonus]
[THE BUTTON — action verb + outcome]
```

```
[ START FREE — NO CARD REQUIRED ]
```

## The FAQ

**Hormozi:** the FAQ section is where you kill the remaining objections. Each Q = an objection the prospect has but didn't articulate. Each A = a one-paragraph reversal.

CombatCall FAQ should hit:
- "Will sportsbooks limit me?" (Probably not at modest stake sizes; we have a guide.)
- "What's the actual win rate?" (Public ledger; here's the link.)
- "Why $120/yr — what makes it worth it?" (Math: if the model gets you +2% better closing-line value on $5K of action per year, that's $100. Most subscribers profit much more.)
- "Can I cancel anytime?" (Yes, Stripe Customer Portal, one click.)
- "What if I'm in WA state?" (We geo-restrict per state laws. See terms.)
- "I'm a casual bettor — is this for me?" (Yes — even one good pick per card pays for the month.)

## The Visual Hierarchy Rule

A great sales page works at three reading speeds:
1. **Headlines + bullets + button only (10 seconds).** Does the offer come through? If no, restructure.
2. **Plus subheads + lead + stack (60 seconds).** Does the mechanism + value land? If no, rewrite.
3. **Full read (5-15 minutes).** Does the story + body copy build belief? If no, the copy isn't doing its job.

Most prospects read at speed 1. Some read at speed 2. The ones who read at speed 3 are the buyers. Optimize all three.
