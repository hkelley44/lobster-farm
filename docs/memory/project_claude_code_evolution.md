name: Claude Code feature tracking
description: LobsterFarm should evolve with Claude Code - new features/skills/commands should be integrated as they ship
type: project
---

Claude Code is constantly evolving. LobsterFarm config and SOPs should be updated as new features, skills, and slash commands are released. Examples mentioned: `/security review` could be part of code review process, `/simplify` could be part of PR submission workflow.

**Why:** The platform is built entirely on Claude Code as sole runtime. Leveraging new capabilities directly improves the platform's output quality.

**How to apply:** When building SOPs and workflows, design them to be extensible so new Claude Code features can be slotted in. Keep an eye on Claude Code changelogs.
